## Notes
- 项目类型: React + Tailwind CSS + Vite + TypeScript
- 核心特点: 纯黑白、无圆角、点阵纹理、"Crunchy"交互
- 数据存储: localStorage (无后端),自动保存机制
- 字体: VT323 (主要) + Press Start 2P (备用),强制应用到 placeholder
- 光标: 自定义像素风格光标 (SVG Data URI)
- 布局: CSS Grid 网格布局,桌面端一行三列,提高信息密度
- 排序逻辑: 欢迎卡片(id: init_01)始终在最左边第一个位置,新建提示词添加到列表末尾
- 拖拽功能: 使用 @dnd-kit 实现,长按拖拽(移动8px激活),欢迎卡片不可拖拽
- 隐形拖拽: 卡片顶部 8px 极细感应区,完全透明,悬停仅显示 1px 虚线和 grab 光标
- 音效系统: Web Audio API 生成像素风音效(Tick/Clack/Copy),音量微弱(0.03-0.06),静默失败不影响体验
- 搜索功能: 图标按钮触发浮动搜索框,支持中英文,匹配标题/描述/内容
- 数据管理: [ DATA ] 菜单统一管理,粘贴导入、导出备份、清空全部三个选项
- 反馈机制: PixelToast 黑底白字提示,瞬间出现停留 2 秒后消失
- 安全机制: 清空全部需要二次确认,红色警告提示
- Markdown 解析: 智能识别分隔符,只有 --- 前后有空行且后面紧跟 # 标题时才拆分,正文中的 --- 和 # 完全保留
- 界面精致度: Header 所有按钮高度统一,使用 py-1.5 保持一致
- 输入体验: 新建卡片时输入框为空,使用 placeholder 显示提示文字
- RPG 风格: 输入框 placeholder 采用冒险游戏风格文案(无 Emoji),Focus 状态显示虚线边框
- Markdown 支持: 导出为 Markdown 格式(# 标题, > // 描述, --- 分隔),支持文本粘贴导入解析
- 所有功能已完成并通过 Lint 检查
